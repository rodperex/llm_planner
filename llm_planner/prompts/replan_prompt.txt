A robot failed a step while executing a plan. You must generate a NEW plan to reach the original goal.

     navigate at all or cannot use any spatial capability.
  2. Rethink the goal achievement strategy from scratch given the new constraint.
     Find a genuinely different approach that avoids only the confirmed missing capability.
     The alternative MUST be concrete and specific — never write vague phrases like
     "using a different method" or "try another approach". Name the exact alternative
     technique, sensor, or action the robot will use instead.
  3. The replan MUST cover ALL sub-goals of the original goal. Do not produce an incomplete
     plan that addresses only the failed step — every objective in the original goal must
     still be achieved (or explicitly declared unreachable with justification).
  4. If the goal is truly unreachable without the failed capability, state it clearly in
     context and produce the best possible partial plan that makes the most progress.
  5. Do NOT repeat already-completed steps.
  6. If the input includes PREVIOUS FAILED STRATEGIES FOR THIS STEP, treat each listed
     strategy as a BLOCKED CAPABILITY — interpret it BROADLY. For example, if a previous
     attempt used navigation and it failed, treat ALL navigation-based approaches as
     unavailable, not just the specific path or target. Each replan must use a genuinely

  8. Before acting on an entity (person, object, location), include a PERCEPTION or
